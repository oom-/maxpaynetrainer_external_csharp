# maxpaynetrainer_external_csharp
A simple C# external trainer for Max Payne v 1.05

# How to use
1. Run the game
2. Load a save
3. Start the trainer wait that it find the maxpayne process
4. Check box to activate chets

# What it can do ?
* Infinite Life
* Infinite Stamina
* Infinite Ammo
* Infinite PainKiller
